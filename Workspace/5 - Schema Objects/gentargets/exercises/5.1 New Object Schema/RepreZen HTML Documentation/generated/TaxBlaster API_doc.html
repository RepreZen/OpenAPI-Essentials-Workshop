Exercise: New Object Schema

In this exercise, we’ll create a new object-typed schema.
The new /tax_payment/{id} path item has a GET method with a 200 response. The schema of the 200 response body only specifies that the value must be an object.

Add the necessary elements to the Schema Object to make it validate a message payload like this:

{
                          "paymentID": "P12347",
                          "date": "2018-04-10",
                          "amount": 2231.89,
                          "currency": "USD"
                        }
